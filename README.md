# Minecraft_Server_Launcher_V2

<img align="left" src="https://github.com/myne145/Minecraft_Server_Launcher_V2/assets/78707580/e7d36409-8678-48e9-9a65-b04bfc8b4319"><br><br>
Minecraft server GUI launcher, intended for better server management in one place.

<br><br><br><br>
# Features
- Built-in server console with commands <img width="900" src="https://github.com/myne145/Minecraft_Server_Launcher_V2/assets/78707580/fbcef93b-5af7-44b7-9a1e-3c26fbcfed92"><br><br>
- Support for adding a Minecraft world to the server from an archive or a folder <img width="900" src="https://github.com/myne145/Minecraft_Server_Launcher_V2/assets/78707580/896bb91b-74e7-4d2a-8b57-fc081139c82a"><br><br>
- CPU & RAM monitoring
- Runs on all platforms (cross-platform)

# Features currently in progress:
- Ability to add and modify servers in-app
- Support for the server.properties file
- Auto-updater
